## Research-Oriented Engineering Projects

### 1️⃣ [Automatic_Generic_Optimizer](https://github.com/datafox-tw/Automatic_Generic_Optimizer)

*A genetic-search framework for structured LLM reasoning optimization.*

This project replaces adversarial debate-style optimization with an **evolutionary search paradigm**:

* Defines structured reasoning outputs as “genomes”
* Implements crossover and mutation strategies on reasoning traces
* Uses fitness scoring to select and refine candidate solutions
* Explores search-based optimization instead of single-pass prompting

Focus areas:

* Search space design for LLM reasoning
* Mutation operators on prompts and reasoning steps
* Fitness scoring and solution aggregation
* Reproducible evaluation workflows

This project reflects my core research direction:
**LLM reasoning as a search and optimization problem.**

---

### 2️⃣ [Chinese_RAG_Expert_Finetuning_System](https://github.com/datafox-tw/Chinese_RAG_Expert_Finetuning_System)

*A reproducible training system for domain-adaptive RAG pipelines.*

Originally developed from a retrieval-focused research task (Recall@10, MRR@10), this project was redesigned into a modular system that:

* Fine-tunes open-source LLMs (e.g., Qwen, LLaMA)
* Optimizes retriever and reranker components
* Supports structured data formatting and repeatable training runs
* Integrates a front-end interface for experiment management

Key emphasis:

* Reproducibility over one-off experimentation
* Modular retriever/reranker architecture
* Evaluation-driven optimization

This project demonstrates my ability to turn research objectives into **system-level implementations**.

### 3️⃣ [GLoVE (GARCH-informed Loss for Volatility Estimation)](https://github.com/datafox-tw/GLoVE)

*A hybrid econometric–deep learning pipeline for multi-asset volatility forecasting.*

This project integrates:

* Classical financial models (GARCH family)
* Neural architectures (e.g., TSMixer)
* End-to-end training and evaluation pipeline
* Clear re-training and replication workflow documentation

Focus areas:

* Multi-asset risk modeling
* Model fusion between econometrics and neural networks
* Structured experiment design
* Reproducible financial ML pipeline

This work reflects my long-term interest in **LLM × Finance × Structured Modeling**.
